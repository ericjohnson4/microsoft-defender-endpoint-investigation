# Microsoft Defender for Endpoint Detection & Incident Investigation

## Project Overview

This project demonstrates the use of Microsoft Defender for Endpoint (MDE) to investigate security alerts, analyze endpoint telemetry, perform threat hunting, and document incident response activities.

The investigation focused on identifying suspicious PowerShell activity, reviewing device timelines, validating alert evidence, and leveraging Microsoft Defender's Advanced Hunting capabilities to support incident analysis.

---

## Objectives

- Investigate Microsoft Defender for Endpoint security alerts
- Analyze suspicious PowerShell execution activity
- Review device timelines and process execution chains
- Utilize Advanced Hunting with KQL
- Document findings and incident response actions
- Demonstrate practical SOC analyst investigation techniques

---

## Technologies Used

- Microsoft Defender for Endpoint (MDE)
- Microsoft Defender XDR
- Microsoft Security Portal
- Advanced Hunting
- Kusto Query Language (KQL)
- Windows Endpoint Telemetry

## Investigation Workflow
